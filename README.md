# Personal Portfolio Website

A modern, responsive portfolio website built with HTML, CSS, and JavaScript. This portfolio showcases professional work experience, projects, and contact information for a graphic designer specializing in branding and digital creative work.

## Features

- **Responsive Design**: Optimized for desktop, tablet, and mobile devices
- **Interactive Elements**: Custom cursor, scroll progress bar, and smooth animations
- **Work Filtering**: Filter work experience by categories (Branding, E-Commerce, Creative Lead)
- **Typing Animation**: Dynamic text animation in the hero section
- **Counter Animations**: Animated statistics on scroll
- **Mobile Menu**: Hamburger menu for mobile navigation
- **Back to Top**: Smooth scroll to top functionality

## Sections

1. **Hero**: Welcome section with animated statistics and call-to-action buttons
2. **About**: Personal information, expertise, and software proficiency
3. **Work Experience**: Professional work history with filtering capabilities
4. **Projects**: Freelance project showcase
5. **Contact**: Contact information and social links

## Technologies Used

- HTML5
- CSS3 (with CSS Grid and Flexbox)
- Vanilla JavaScript
- Google Fonts (Inter)
- Font Awesome icons (via Unicode)

## How to View

Simply open `index.html` in any modern web browser. No server required as it's a static website.

## Customization

To customize this portfolio for your own use:

1. Update personal information in the HTML (name, bio, contact details)
2. Replace work experience and project details
3. Modify the color scheme in the CSS variables (root section)
4. Add your own images by replacing the emoji placeholders
5. Update social media links and contact information
